## SYNOPSIS
Active Directory helpers for locked accounts and password operations.

## SYNTAX

```
Get-LockedADAccounts
```

## DESCRIPTION
Provides minimal functions for retrieving locked-out users, unlocking
accounts, and resetting passwords with an optional "change at next logon".
