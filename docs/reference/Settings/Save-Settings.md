# Save-Settings

## SYNOPSIS
Persists settings to local and, when shared is available and newer, to the preferred shared path.

## SYNTAX

```
Save-Settings [-Settings] <Object> [-ScriptPath] <String> [<CommonParameters>]
```

## DESCRIPTION
Always writes sanitized settings to %ProgramData%\ADapp\Data\settings.json (local). When Data.PreferredSharedPath is configured and accessible, pushes to that shared path if the local file is newer than the shared file or the shared file does not exist. Secrets are persisted to %ProgramData%\ADapp\config\settings.local.json. App-dir config/settings.json is read-only and is not updated. Triggers Sync-SharedData when available so other data (favorites, history) stays in sync.
