# Load-Settings

## SYNOPSIS
Loads merged application settings with shared-first behavior and key-preserving normalization.

## SYNTAX

```
Load-Settings [-ScriptPath] <String> [<CommonParameters>]
```

## DESCRIPTION
Uses shared settings as the source when available: resolves the active path from Data.PreferredSharedPath (if set and accessible) or %ProgramData%\ADapp\Data, then loads that path's settings.json and merges it over app config/settings.json defaults. Falls back to app config when shared is not set or inaccessible. Merges %ProgramData%\ADapp\config\settings.local.json for secrets, then applies ADAPP_* environment overrides. Normalization validates known keys but preserves unknown keys so custom settings are not dropped.
