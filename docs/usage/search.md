# Search

## Basic Search

1. Enter a search term in the search box
2. Select object type: All, User, or Computer
3. Click Search or press Enter

Results appear in the upper ListView with columns for Type, Name, Details, Status, Last Logon, and Pwd Last Set.

## Advanced Tips

- Use wildcards to broaden queries (e.g., `sm*`)
- Searching for users supports "First Last" or "Last First" formats
- Sort any column by clicking its header; click again to toggle ascending/descending

## Filtering Suggestions

- Filter by object type via the Type dropdown
- Use the status column to visually identify Offline/Disabled entries
- Use Favorites and History (Search menu) to reuse common queries
