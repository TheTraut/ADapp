## Keyboard Shortcuts

- Default bindings:
  - Focus search: Ctrl+F
  - New search: Ctrl+N
  - Clear filters: Esc
  - Remote Desktop: Ctrl+Alt+R
  - VNC Connection: Ctrl+Alt+V

- Change bindings via Settings â†’ Shortcuts. Click in a field and press the desired key combo.

- Shortcuts are app-wide while the main window is focused.

- Settings are stored in the active shared data folder and synced between machines when a preferred shared path is configured.
