# BitLocker

## Retrieve Recovery Password

1. Select a computer from search results
2. Use the AD → Computer → BitLocker Recovery menu option
3. Follow prompts to retrieve the recovery key

## Notes

- Requires appropriate permissions to read BitLocker recovery information from AD
- Log access attempts for auditing where possible
