# Computers

## View Details

- Select a computer to populate the details panel
- Online/Offline is indicated; password last set and last logon are shown when available

## Remote Access

- RDP: Launch Remote Desktop
- VNC: Launch VNC viewer (path configured in `config/settings.json`)
- File Explorer: Open administrative shares or file paths
- Computer Management: Open MMC Computer Management console

## Power Management

- Shutdown/Restart where permitted

## Tips

- Use the Status column and color cues to spot Offline systems
- Configure `VNCPath` in `config/settings.json` for VNC integration
