# Favorites & History

## Favorites

- Save common searches to the Favorites menu
- Quickly recall pre-configured queries without retyping filters

## History

- View recent searches in the History menu
- Re-run a prior search by selecting it from the list

## Tips

- Keep Favorites for stable, frequently used queries
- Use History for ad hoc recall while iterating
