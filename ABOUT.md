# Active Directory Management Tool

Version: 1.0.16

## Description
A comprehensive Active Directory management tool built with PowerShell and Windows Forms. This application provides an intuitive interface for managing users, computers, groups, and other AD objects.

## Features
- Advanced search capabilities with filtering options
- Support for searching users by first and last name in either order
- Group membership management
- Computer management tools (RDP, VNC, File Explorer)
- User account management (password reset, enable/disable, unlock)
- BitLocker recovery password retrieval
- Favorites and search history
- User location tracking
- Comprehensive logging and error handling

## Requirements
- Windows 10 or Windows Server 2016 (or newer)
- PowerShell 5.1 or later
- .NET Framework 4.7.2 or newer
- Active Directory module
- Windows Forms assemblies
- Administrative privileges for AD operations
